# Assets de marca

Coloca aqui el logo de Win para el header del correo:

- Archivo: `win_logo.png`
- Version recomendada: logo **WIN en blanco + claim "EL INTERNET DE LOS WINNERS"**,

Si el archivo no existe, el correo usa un wordmark de texto "win" como respaldo.
